# JavaScript

##### 8 data types

1. `undefined`
2. `null`
3. `boolean`
4. `string`
5. `symbol`
6. `bigint`
7. `number`
8. `object`

##### Declare variables

```js
var variableA;
const variableB;
```

##### Assignment

```js
variableA = 10;
var variableB = 10;
```

**Escaping Literal Quotes in Strings**

quote 앞에 backslash (`\`)를 붙이면 사용할 수 있다.

```js
var sample = "I said \"javascript\", and they ran away."
```
